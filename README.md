# ANIME CRUD — FPY1101

Programa en Python que implementa las operaciones CRUD
sobre uno(s) anime(s) usando funciones. 
Añadiendo nombre del anime, genero de este, capitulos y temporadas pudiendo editarlo y eliminar en caso de ser necesario almacenando todo en un diccionario.
Evitando problemas de posible caida.

## Funciones
- `agregar_producto(Nombre, Genero, Capitulos y Termporadas)` — CREATE
- `mostrar_anime()` — READ
- `editar_anime(Nombre, Genero, Capitulos y Temporadas)` — UPDATE
- `eliminar_anime()` — DELETE
- `mostrar_anime()` — DELETE

## Requisitos Técnicos Aplicados
- Uso de listas y diccionarios para almacenar los datos proporcionados por el usuario.
- Uso de funciones documentadas para las distintas necesidades del mismo codigo segun vaya avanzando "el añadido de nombre, genero, capitulos y temporadas".
- Validaciones y manejo de excepciones (`try/except`) para evitar posibles caidas del codigo "sabiendo" que hacer en caso de encontrarse con una aplicacion de datos cuestionables como puede ser... Escribir un numero "Uno" en vez de "1" o distintas formas de escribir una palabra y que no impida su correcto funcionamiento.
